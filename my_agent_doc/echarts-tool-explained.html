第 2 步 · 一张图看懂流程

从"数据 + 一句话"，到"一张图"

这个工具嵌在 Agent 身上——Agent 一识别到有画图需求，就自动调用它。内部分两步：选图小专家（调 LLM，只看抽样几行）定出图规则，再由程序按规则解析全量数据出图。

第 3 步 · 三个亮点

它比"画个图"多做了什么

🧠

自动选图

选图小专家（LLM）只看抽样几行 + 意图，就输出"出图规则"：用什么图、X/Y 怎么放、要不要分系列。说人话就行，不用人指定。

🛡️

稳：失败有兜底

推断最多重试 3 次；再不行用规则兜底（默认第一列做 X、数值列做 Y、柱状图），保证总出得了图，不会画不出来。

🗃️

不撑爆上下文

LLM 选图只喂抽样，全量数据由程序处理、不塞进 LLM 上下文——又快，又不会撑爆模型记忆或被截断。
